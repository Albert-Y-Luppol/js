template expression operators

Pipe Ex: {{ "Hello" | uppercase }} {{ "Hello" | lowercase }} {{ date | date: "longDate" }} {{ { name: "Tom", surname: "Souer", age: 10 } | json }}

Safe "?." symbol: {{ { val1: 1, val2: 2 }?.name }} {{ { val1: 1, val2: 2 }.name }} {{ { val1: 1, val2: 2, name: null }.name?.amount }}

Non-null accertion operator "!.": {{ "null"!.toUpperCase() }} {{ { name: "tom" }.name!.toUpperCase() }}

The item's undeclared best by date is: {{ $any(item).bestByDate }}
